Simple Modal Window - Don't let your dreams be dreams

As my first blog post I would like to highlight something which I was very happy to have accomplished. While its no towering technical feat, creating a nice looking modal window (not sure if thats what its called) was something that, only a few months ago, seemed to be completely out of reach. So, without any further ado, I bring you my simple yet effective approach for taming this mighty foe.

First things first. We are going to need a few CDNs to get the ball rolling. Specifically Angular, Angular-animate, and the Anguluar UI BootStrap. So go ahead and copy those over.

Next, in the angular controller responsible for whatever element you are going to want to start this task off of, we are going to create a function that will allow the modal instance to open. All we need to do is instruct the uibModalInstance to open, and then assign a templateUrl where we will place our html/javascript that will be seen in the window, and then assign a controller that will have authority over it

As for the templateURL, what we are going to want to do here is create a script element where type = text/ng-template. This allows angular to process it so it can be used for ngView, ngInclude, or in this case, our uibModalInstance. We can now treat everything inside that script tag as if it were an html document.

Last thing we are going to do here is create a function that will allow the modal to be closed. What I did was simply create a button that when clicked, will trigger a function that was declared inside the controller we specificed when we opened the modal instance. We simply call "$uibModalInstance.close();" and we are done.
